# 💄 Beauty Academy - Event Management System
**Ujian Tengah Semester - Pemrograman Web 2**

Beauty Academy adalah platform manajemen workshop kecantikan (Makeup, Skincare, Hairdo) yang memungkinkan admin untuk mengelola data kategori, pembicara, dan event secara terintegrasi. Project ini dibangun menggunakan stack modern Full-stack TypeScript.

## 🚀 Fitur Utama
- **Landing Page**: Beranda estetik bertema Beauty Class dengan informasi Seminar, Workshop, dan Talkshow.
- **Authentication**: Login khusus Admin menggunakan NIM Mahasiswa.
- **Dashboard Admin**: Ringkasan data (Statistik) sederhana.
- **Manajemen Kategori**: Menambah, melihat, dan menghapus kategori kegiatan.
- **Manajemen Speaker**: Mengelola narasumber/pakar kecantikan.
- **Manajemen Event**: Input event dengan fitur **Dropdown Dinamis** (mengambil data kategori & speaker dari database).
- **Menu Biodata**: Halaman profil pengembang website.

---

## 🛠️ Tech Stack
- **Frontend**: React (Vite), TypeScript, Tailwind CSS, Lucide Icons.
- **State Management**: Zustand (Persist Middleware).
- **Form Handling**: React Hook Form & Zod (Validation).
- **Backend**: Express.js, TypeScript.
- **Database**: Prisma ORM & Supabase (PostgreSQL).
- **Deployment**: Vercel & GitHub.
